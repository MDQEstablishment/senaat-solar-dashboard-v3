# SENAAT Solar Dashboard v3

Single-file React HTML prototype for the SENAAT Solar Projects Dashboard.

## Files
- `SENAAT_Solar_Dashboard_v3.html` — open in any modern browser (no build step)
- `REQUIREMENTS.md` — functional spec
- `HANDOFF.md` — project context and work plan

## Stack
- React 18 + ReactDOM (UMD via CDN)
- Tailwind CSS via CDN
- Recharts via CDN
- Babel standalone for inline JSX

## Features
- 4 distinct role views (VP / Program Manager / Project Manager / Site Engineer)
- 8 programs covering 1,600 schools
- 12-stage execution lifecycle (editable in Settings)
- School detail with stage timeline + chat panel
- Tasks, notifications, escalations
- Settings: Lifecycle, Stage Statuses, Custom Fields, Milestone Templates
- Financials with auto-rollup from entries table
- Recycle Bin (soft-delete with restore)
- Materials, Contractors, Reports
